DOM (Document Object Model)

DOM is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree-like data structure, where each element in the document is a node in the tree. DOM allows you to access, manipulate, and update the content and structure of a web page dynamically using JavaScript.

1. DOM Tree Structure

The DOM tree is a hierarchical representation of an HTML or XML document. It starts with the root node (usually <html>), and all other elements, attributes, and text nodes are its descendants.

For example, consider the following HTML snippet:

<!DOCTYPE html>
<html>
<head>
    <title>DOM Example</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
</body>
</html>

The DOM tree structure for this HTML would look like:

Document
└── html
    ├── head
    │   └── title
    │       └── "DOM Example"
    └── body
        ├── h1
        │   └── "Hello, World!"
        └── p
            └── "This is a paragraph."

2. DOM Nodes

DOM nodes are the elements, attributes, and text content in the DOM tree. Each node can be an element node, attribute node, or text node.

  • Element Node: Represents HTML elements, such as <div>, <p>, <h1>, etc.
  • Attribute Node: Represents attributes of an element, such as src, href, etc.
  • Text Node: Represents the text content within an element.

3. Accessing DOM Elements

You can access DOM elements using JavaScript by using various methods, such as:

  • document.getElementById(): Returns a single element with the specified ID.
  • document.getElementsByClassName(): Returns a collection of elements with the specified class name.
  • document.getElementsByTagName(): Returns a collection of elements with the specified tag name.
  • document.querySelector(): Returns the first element that matches the specified CSS selector.
  • document.querySelectorAll(): Returns a list of all elements that match the specified CSS selector.

Example:

<div id="myDiv">This is a div.</div>
<p class="myClass">This is a paragraph.</p>

<script>
    const divElement = document.getElementById("myDiv");
    const pElements = document.getElementsByClassName("myClass");
</script>

4. Manipulating DOM Elements

You can manipulate DOM elements by changing their content, attributes, or style.

  • Changing Content: Use textContent, innerHTML, or innerText properties.
  • Changing Attributes: Use setAttribute() method or directly access the attribute using the element's properties (e.g., element.src, element.href).
  • Changing Style: Use style property to access and modify CSS properties of an element.

Example:

<p id="myPara">This is a paragraph.</p>

<script>
    const paraElement = document.getElementById("myPara");
    paraElement.textContent = "This is updated content.";
    paraElement.setAttribute("class", "newClass");
    paraElement.style.color = "blue";
</script>

5. Creating and Appending Elements

You can create new elements dynamically and append them to the DOM.

  • document.createElement(): Creates a new element node.
  • element.appendChild(): Appends a child node to the specified parent element.

Example:

<div id="myDiv">This is a div.</div>

<script>
    const divElement = document.getElementById("myDiv");
    const newPara = document.createElement("p");
    newPara.textContent = "This is a new paragraph.";
    divElement.appendChild(newPara);
</script>

6. Event Handling

DOM allows you to attach event listeners to elements to respond to user interactions, such as clicks, keypresses, etc.

Example:

<button id="myButton">Click Me</button>

<script>
    const button = document.getElementById("myButton");
    button.addEventListener("click", function() {
        alert("Button clicked!");
    });
</script>

7. DOM Traversal

DOM traversal involves navigating between nodes in the DOM tree, moving from parent to child or from child to parent.

Example:

<ul id="myList">
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

<script>
    const myList = document.getElementById("myList");
    const listItems = myList.getElementsByTagName("li");

    for (let i = 0; i < listItems.length; i++) {
        console.log(listItems[i].textContent);
    }
</script>
